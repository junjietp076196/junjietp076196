Enterprise Network Design Project

Cisco Packet Tracer VLAN & OSPF HSRP Redundancy Network Security

[cite_start]A scalable, redundant, and secure network infrastructure designed for a multi-department enterprise, featuring advanced switching and routing protocols[cite: 1, 3].

Project Overview

This project involved designing and configuring a complete network topology for an organization with multiple departments (Sales, HR, IT, Finance). The goal was to ensure efficient data flow, high availability, and robust security using Cisco networking equipment.

[cite_start]The design utilizes Variable Length Subnet Masking (VLSM) to minimize IP address wastage and OSPF for dynamic routing between different branch locations[cite: 3, 13].

Key Configuration & Features

1. Network Segmentation & Security

    [cite_start]
  • VLANs: Segregated traffic into distinct VLANs (Sales, HR, IT, Finance, Server, Wireless) to improve performance and security[cite: 8].
    • [cite_start]
  • Port Security: Implemented "Sticky" MAC address learning on access ports, restricting access to a maximum of 2 devices per port to prevent unauthorized connections[cite: 10].
    • [cite_start]
  • SSH Access: Configured secure remote management for all devices[cite: 5].

2. High Availability & Redundancy

Ensuring the network stays online even if a device fails.

  • HSRP (Hot Standby Router Protocol): Configured Active and Standby routers to provide a redundant default gateway. [cite_start]If the main router fails, traffic automatically switches to the backup[cite: 17].
    • [cite_start]
  • EtherChannel: Bundled physical links into a single logical channel using LACP, increasing bandwidth and providing link redundancy between switches[cite: 16].

3. Routing & Services

    [cite_start]
  • Inter-VLAN Routing: Used Router-on-a-Stick configuration with subinterfaces to allow communication between different departments[cite: 13].
    • [cite_start]
  • DHCPv4: configured routers to act as DHCP servers, automatically assigning IP addresses to end devices[cite: 19].
    • [cite_start]
  • Wireless LAN: Deployed a Wireless LAN Controller (WLC) to manage Lightweight Access Points (LAPs) for secure Wi-Fi connectivity[cite: 20].

Challenges & Learnings

Configuring EtherChannel presented challenges with mismatched port protocols. I learned the importance of consistent configuration (speed, duplex, VLAN modes) across both ends of a link. [cite_start]Additionally, calculating the VLSM scheme required precision to ensure no IP conflicts occurred between the 13 different subnets required[cite: 3].
